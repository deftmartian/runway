runway is a private, self-hosted running plan and activity review app.

On first launch, choose your self-hosted runway server. The app verifies that address, then opens the
complete runway PWA with the server origin visible in a browser Custom Tab. Planning, calendar, run
detail, import review, history, stats, settings, and authentication are all part of the installed app
experience. runway does not hide a user-entered server in an embedded WebView.

Android also retains read access to a user-approved Gadgetbridge GPX export folder and can schedule
bounded checks without broad storage, location, fitness, Play Services, or advertising permissions.
You can share one GPX file to runway from another Android app.

Native folder and share imports use a scoped device credential, stable request ids, bounded reads,
content-based revision checks, and the same review inbox as browser imports. Folder access remains
limited to the directory you choose, and imported activities never change the plan before review.
